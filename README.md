# Vinit Patel — Portfolio

Static wildlife & corporate photography portfolio.

## Deployment

Connected to Vercel via GitHub. Every push to `main` auto-deploys.

## Updating the site

Drop the new `index.html` into this folder, then run:

```bash
git add index.html
git commit -m "update"
git push
```

Vercel picks it up automatically — no link change needed.
